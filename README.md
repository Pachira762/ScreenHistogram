# ScreenHistogram
スクリーン上の任意の範囲からヒストグラムが取れるアプリケーションです。  
https://user-images.githubusercontent.com/34003267/146632369-b383bf44-6d46-4117-833d-ea4afdbc65f1.mp4


## ダウンロード
https://github.com/Pachira762/ScreenHistogram/releases  

## 機能
RGB, Lumaのヒストグラム  
輝度、彩度の可視化  
カラーピッカー
